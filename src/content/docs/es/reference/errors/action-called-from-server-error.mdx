> **ActionCalledFromServerError**: Acción llamada desde una página renderizada en el servidor o endpoint sin usar `Astro.callAction()`.

## ¿Qué ha ido mal?
Acción llamada desde una página renderizada en el servidor o endpoint sin usar `Astro.callAction()`.
